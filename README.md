# GMT Iseng 🚀

Project Django iseng dengan gaya minimalis-modern-elegant. Dibuat buat belajar Django sambil ngobrol dengan AI.

## ✨ Fitur
- Halaman login dengan split screen
- Gradien Cobalt Blue & Burnt Orange (warna favorit)
- Redirect dari home ke login
- Django authentication siap pakai

## 🛠️ Teknologi
- Django 6.0.2
- CSS Manual (No Tailwind ribet)
- Font Inter from Google Fonts

## 🚀 Cara Jalanin

```bash
# Clone repo
git clone https://github.com/dudigmt/gmt-iseng.git
cd gmt-iseng

# Buat virtual environment
python3 -m venv .venv
source .venv/bin/activate  # Linux/Mac
# atau .venv\Scripts\activate  # Windows

# Install dependencies
pip install django

# Migrasi database
python manage.py migrate

# Buat superuser (opsional)
python manage.py createsuperuser

# Jalanin server
python manage.py runserver 8008

Buka http://127.0.0.1:8008 dan nikmati.
